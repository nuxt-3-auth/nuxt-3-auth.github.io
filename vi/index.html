Nuxt 3 Auth Module

A simple authentication module for Nuxt 3

Bắt đầu
API Reference

Đơn giản

Đơn giản, nhỏ gọn nhưng đầy đủ những tính năng cơ bản của một module authentication.

Tương thích cao

Được thiết kế để làm việc với các module phổ biến của Nuxt như i18n, ofetch, pinia.

Sẵn sàng sử dụng

Có sẵn các trang đăng nhập, đăng xuất, form đăng nhập, store quản lý session, và các composition apis dễ dàng tùy chỉnh.

Hỗ trợ refresh token

Hỗ trợ refresh token, tự động refresh access token khi hết hạn. Hỗ trợ chức năng ghi nhớ đăng nhập.
